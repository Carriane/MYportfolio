Temperature Conversion

This Ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding temperature is in Fahrenheit. 


      puts "Enter degrees in Celsius: "
      num = gets.to_i

      val = (1.8 * num) + 32
      puts "It is #{val.to_i} degrees Fahrenheit!"
      


      puts "Enter a number"
      user_number = gets.to_i
      last_digit = user_number % 10 

      if  last_digit == 1 && user_number != 11
      puts "#{user_number}st place"
      elsif  last_digit == 2 && user_number != 12
      puts  "#{user_number}nd place"
      elsif last_digit  == 3 && user_number != 13
      puts "#{user_number}rd place"
      elsif last_digit == 11 || 12 || 13 
      puts "#{user_number}th place"
      else 
      puts "#{user_number}th place"
      end

Foobar

This Ruby program is an adaptation og a classic technical interview problem which displays sequences of the Foobar pattern. 


        puts "How many numbers do you want to see?"
        input = gets.chomp.to_i

        numbers = []
        (1..input).each do |num|

        if num % 3 == 0 && num % 5 == 0
        numbers.push("foobar")
        elsif num % 3 == 0 
        numbers.push("foo")
        elsif num % 5 == 0
        numbers.push("bar")
        else 
        numbers.push(num)
        end
        end
        puts numbers
